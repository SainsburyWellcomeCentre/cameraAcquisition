Requires Bonsai 2.9.0 with PointGrey, Scripting.expressions, Vision, Dsp packages. 
Requires Spinnaker SDK 4.2.0.83.
Use launch_bonsai.m if running within the experiment protocol on MATLAB. 
If running separately, change paths in launch_cameras.py and run in the terminal:
```
python launch_cameras.py
```
